About the Project

Our goals, approach, and what we learned.

Our Goal

Build a fully functional, multi-page static website that demonstrates core web development skills in HTML, CSS, and JavaScript.

Our Approach

We divided responsibilities by skillset, held weekly check-ins, and used version control to coordinate our work effectively.

What We Learned

From semantic HTML and responsive CSS to form validation and team communication — this project was a hands-on learning experience.
