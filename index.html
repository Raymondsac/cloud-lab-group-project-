Project Description

This website was created as a group assignment for the Web Technologies course. The goal is to demonstrate practical skills in HTML, CSS, and JavaScript by building a multi-page static website that is well-structured, visually consistent, and easy to navigate. The project covers semantic markup, responsive layouts, form handling, and basic interactivity.
